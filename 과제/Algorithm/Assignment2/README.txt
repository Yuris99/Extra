
GCC 환경에서 다음과 같이 실행 (과제 채점할 때도 아래와 같이 실행하므로 참고)
소스코드 파일명은 반드시 학번.c (또는 학번.cpp)로 할 것.

제공되는 입력 형식
 - 파일명 형식 : input.txt
 - 데이터 형식 : input.txt 확인

출력 형식
  1. option "array"
    - 파일명 : example_output_array.txt 확인
  2. option "path"
    - 파일명 : example_output_path.txt 확인

* C 언어
   - 소스코드 파일명 : main.c
   - 소스코드 컴파일 : gcc main.c -o run.exe
   - 실행 : ./run.exe [array 또는 path] < input.txt

* C++ 언어
   - 소스코드 파일명 : main.cpp
   - 소스코드 컴파일 : g++ main.cpp -o run.exe
   - 실행 : ./run.exe [array 또는 path] < input.txt
